# Series: Design Requirement Summary for Khel Tic Tac Toe

How to use design memory to create themed wireframes, with the core layout remaining unchanged through prompts.

We start with creating wireframes using a particular theme and core elements for applications. However, when themes change, keeping the core same with just change of themes can be mortifying- new prompts, multiple iterations, lost design memory.

This series is an account on how i dealt with this issue while experimenting designs for a simple gaming application.
