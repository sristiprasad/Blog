# Series: How managing with generative AI is a pain- Google Stitch

I was using google stitch to create a set of wireframes for a mobile gaming app.
The start was great! it took all my broken prompts to still give asthetic results.
First 7 set of wireframes with matching theme and maintaining UI cohesiveness- it was like hanging out in a garden! UNTIL....

Gen-AI started hallucinating when i changed the theme and a few coin designs.

This series is an account on how i failed with generative AI tool google stitch and ended up spending over 48 hours to produce 14 sets of wireframes while experimenting designs for a simple gaming application.
