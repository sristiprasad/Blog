# Setting up design context

This post cover my initial prompts to create a south indian theme based wireframes for a tic tac toe gaming application. I used google stitch as my AI assistant for achieve this.

I start with a project overview, then dive in directly in design prompting.
Steps for the first 3 screens including master is presented below:-

## Project Overview

A premium, culturally-rooted Tic Tac Toe mobile game featuring high-fidelity, intricate line-art iconography based on South Indian temple architecture and traditional motifs.

## Master Screen: Contexting for core UI elements to be resused in subsequent screens.

### Prompt:

Act as an expert UI designer engineer. Generate a wireframe for the below idea.
Wireframe Idea: Simple structural skeleton design.

Design a wireframe with the following instructions-

1. The layout of the screen should be 9:12 for mobile apps
2. Primary background of wireframe- Use the design colour profile of#0c5021 (deep forest green with slight yellow undertone). Maintain this pallette through the entirety of wifeframe without gradient flush
3. Add a tic tac toe 3x3 grid at the center of the wireframe with the below attributes-Grid Lines should be in Muted Gold ( #C0C0C0 at 40-50% opacity) grid lines should be open and not boxed or confined
4. Below the grid vertically distant ~40-60px from the grid position "KHEL TIC TAC" in all caps, #C9A227 colour and Noto Serif font semi-bolded. Align it in the centre

Do not add anything else on the wireframe not mentioned. no additional lines, headers, taskbars required

Ask any questions for clarity in making the wireframe.

## Screen1: Contexting for splash screen while resusing core UI elements from master screen

### Prompt:

Act as an expert UI designer engineer. Generate a wireframe for the below idea.
Wireframe Idea: Splash Screen for Khel tic tac toe

1. The layout of the screen should be 9:12 for mobile apps
2. Primary background of wireframe- use the design colour profile of #0c5021 (deep forest green with slight yellow undertone). Maintain this pallette through the entirety of wifeframe without gradient flush
3. Add a tic tac toe 3x3 grid at the center of the wireframe with the below attributes- Grid Lines should be in Muted Gold (#C0C0C0 at 40-50% opacity). Grid lines should be open and not boxed or confined.
4. Below the grid vertically distant ~40-60px from the grid position "KHEL TIC TAC" in all caps and #C9A227 colour with Noto Serif font semi-bolded. Keep the font size 20 and fit it in one line only. Align it to the center
5. Add coins in the 3x3 grid as follow:

Coin 1: Form: A symmetrical, designer stepped-pyramid structure inspired by Tirupati temple in #C9A227 . tacked rectangular tiers that decrease in width as they ascend. Pronounced "Kalash" peak at the very top to resemble temples top. Stroke: Thin gold line-art (1.5-2px) with slightly rounded joins. Scale: Occupies ~60-70% of the grid cell. add in grid (1x1, 2x2, 3x3). Make it look intricately designed in an embossed effect.

Coin 2: An abstract, organic spiral motion representing wind or ocean waves in (#C0C0C0) . add in grid (1x3, 2x1, 3x2). make it look intricately designed in an intricate embossed effect

Do not add anything else on the wireframe not mentioned. no additional lines, headers, taskbars required.

## Screen2: Onboarding Screen

### Prompt:

Act as an expert UI designer engineer. Generate a wireframe for the below idea.
Wireframe Idea: Onboarding screen to enter user name

1. The layout of the screen should be 9:12 for mobile apps
2. Primary background of wireframe- use the design colour profile of #0c5021 (deep forest green with slight yellow undertone) .Maintain this pallette through the entirety of wifeframe without gradient flush.
3. Add a small box with bolded border, rounded corners in #D4AF37 and filled in #052B1B. Inside the box aligned to the center include text 'Enter your name ' in #D4AF37  colour. Keep this in semi bolded font Noto Serif.
4. Below this text include another text in #C0C0C0 ' Your Name' indicating the exact position where user must include their name. This should vanish when cursor appears in this area or if use types names. Align this to the centre.
5. Below this include a straight line in #D4AF37, which is the position underlying the typed name.
6. Include a small button which rounded corners and rectangular shape in filled #D4AF37 and text 'Enter' in colour #0C5021 , semi bolded and noto serif font.

Do not add anything else on the wireframe not mentioned. no additional lines, headers, taskbars required. Ask any questions for clarity in making the wireframe.

Wireframe Idea: Homepage with profile information, chat, game statis, settings and game mode transition.

1. The layout of the screen should be 9:12 for mobile apps.
2. Primary background of wireframe- use the design colour profile of  #0c5021 (deep forest green with slight yellow undertone), maintain this palette through the entirety of wireframe without gradient flush.
3. Then, add a tic tac toe 3x3 grid at the exact centre of the wireframe aspect with the below attributes- Grid Lines should be in Muted Gold (#C0C0C0 at 40-50% opacity).  Grid lines should be open and not boxed or confined. Below the grid, vertically distant ~40-60px from the grid position "KHEL TIC TAC" in all caps and #C9A227 colour with Noto Serif font semi-bolded. Make the font size 20 and fit it in one line only.
4. Then add coins in the 3x3 grid as follows:

Coin 1: Form: A symmetrical, designer stepped-pyramid structure inspired by Tirupati temple in #C9A227 . tacked rectangular tiers that decrease in width as they ascend. Pronounced "Kalash" peak at the very top to resemble the temple's top. Stroke: Thin gold line-art (1.5-2px) with slightly rounded joins. Scale: Occupies ~60-70% of the grid cell. add in grid (1x1, 2x2, 3x3). make it look intricately designed in an embossed effect and great detail. Dont make it look kiddish or like a childs drwaing.  

Coin 2: An abstract, organic spiral motion representing wind or ocean waves in (#C0C0C0). add in grid (1x3, 2x1, 3x2). make it look intricately designed in an intricate embossed effect.and great detail. Dont make it look kiddish or like a childs drawaing.  

1. Add a an empty profile display icon in the top left corner of the wireframe aspect.  Beside the profile icon, include the name 'Arjun' in #C9A227 and font noto serif. Keep it bolded.
2. Then on the right corner add a small chat icon, stats icon like a bar chart and a settings icon. maintain the distance between left and right icons.
3. Finally add a small button with rounded corners and rectangular shape in filled #D4AF37 and text 'Lets Play' in colour #0C5021 , semi bolded and noto serif font.

Do not add anything else to the wireframe not mentioned. No additional lines, headers, or taskbars required.
