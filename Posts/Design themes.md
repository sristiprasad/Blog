# Design Themes for Khel Tic Tac Toe

I started exploring the Google stitch tool that claims to replace UI designers by making design ideation and prototyping easy and quick.

However, my experience was laborious when i wanted t create variant theme while keeping core element of this games wireframes intact.

For ex: The homepage of the game will exists for all screens with the following UI elements unaltered:

1. The homepage will consist of a empty profile display icon on top left corner of the wireframe aspect. Next to it the name of the player must appear in a certain font
2. Aligned in the same line on the top right, there would be three icons for chat, game stats in bar char symbol and settings icon
3. A 3x3 unboxed and unbounded grid will appear centrally
4. Coins for the themes will always be in (1x1, 2x2, 3x3) grids and (1x2, 2x3, 3x1) grids
5. Lets play button with text "Lets play" will appear centrally below the grid in rectangular shape

Except the above elements the colour, gradient, coin designs and font schemes keep changing.
